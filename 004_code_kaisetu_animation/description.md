# プログラミングコード解説アニメーションツール

## 概要

このツールは、プログラミングコードとその解説テキストを入力するだけで、コードの特定の部分をハイライトしながら、対応する解説をタイピング風に表示するアニメーションを生成するウェブアプリケーションです。教育的なコンテンツや、コードレビューのプレゼンテーションなどを視覚的に分かりやすく作成するのに役立ちます。

## 主な機能

1.  **コードと解説の入力:**
    *   左側のテキストエリアにプログラミングコードを、右側のテキストエリアにその解説を入力します。
    *   解説は、`##行番号` または `##開始行.終了行` という特別な形式で始めることで、どのコード行に対応する解説なのかを指定できます。

2.  **多言語対応のシンタックスハイライト:**
    *   JavaScript, Python, Java, C#, C++ など、複数のプログラミング言語に対応したシンタックスハイライトが自動で適用され、コードを美しく表示します。
    *   言語はドロップダウンリストから選択できます。

3.  **タイピングアニメーション:**
    *   「開始」ボタンを押すと、指定されたコード行が黄色くハイライトされ、対応する解説が右側のパネルに一文字ずつタイピングされるように表示されます。
    *   タイピング速度はミリ秒単位で調整可能です。

4.  **再生コントロール:**
    *   アニメーションの再生中に「一時停止」「再開」が可能です。
    *   出力画面のどこかをクリックすると、最初の入力画面に戻ります。

## ファイル構成

*   `index.html`: アプリケーションの基本的な構造を定義するHTMLファイルです。コード入力エリア、解説入力エリア、各種設定、および出力画面のコンテナが含まれています。外部ライブラリである`Prism.js`（シンタックスハイライト用）をCDN経由で読み込んでいます。
*   `style.css`: アプリケーションのレイアウトとデザインを指定するCSSファイルです。入力画面と出力画面の2カラムレイアウト、コードと解説パネルのスタイル、ハイライトの色などを定義しています。
*   `script.js`: アプリケーションのすべての動的な機能を制御するJavaScriptファイルです。以下の主要なロジックを実装しています。
    *   入力されたコードと解説テキストの取得と解析。
    *   `Prism.js`を使用したシンタックスハイライトの適用。
    *   解説文の`##`を解釈し、対応するコード行をハイライトする機能。
    *   `setTimeout`を駆使したタイピングアニメーションの実現。
    *   一時停止、再開、リセットなどの再生制御ロジック。
